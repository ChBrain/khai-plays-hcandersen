# The Mute Book: Reference

## Line of Work

Staging Hans Christian Andersen's _Den stumme Bog_ (SDU-097). En studerende begraves med en bog fuld af tørrede blade, hvor hvert blad gemmer på et stumt minde.

## Origin

Hans Christian Andersen, _Den stumme Bog_, catalogued as SDU-097 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Den stumme Bog_ (SDU-097)             | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-097 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
