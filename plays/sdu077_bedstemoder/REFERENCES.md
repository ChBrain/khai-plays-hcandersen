# Grandmother: Reference

## Line of Work

Staging Hans Christian Andersen's _Bedstemoder_ (SDU-077). Bedstemoder sidder i sin lænestol, fuld af kærlighed, men må sige farvel og efterlade minder hos sit barnebarn.

## Origin

Hans Christian Andersen, _Bedstemoder_, catalogued as SDU-077 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Bedstemoder_ (SDU-077)                | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-077 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
