# Skilles og mødes: Reference

## Line of Work

Staging Hans Christian Andersen's _Skilles og mødes_ (SDU-611). En vaudeville i to dele: spanske tropper i Odense 1808, og et gensyn femogtyve år senere.

## Origin

Hans Christian Andersen, _Skilles og mødes_, catalogued as SDU-611 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Skilles og mødes_ (SDU-611)           | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-611 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
