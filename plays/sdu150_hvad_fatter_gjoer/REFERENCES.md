# What the Old Man Does is Always Right: Reference

## Line of Work

Staging Hans Christian Andersen's _Hvad Fatter gør, det er altid det Rigtige_ (SDU-150). En godtroende bonde gør kun dårlige byttehandler, men vinder alligevel guld pga. sin kones urokkelige kærlighed.

## Origin

Hans Christian Andersen, _Hvad Fatter gør, det er altid det Rigtige_, catalogued as SDU-150 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                                      | Scope                                                                 |
| ----------------- | ----------------------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Hvad Fatter gør, det er altid det Rigtige_ (SDU-150) | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk)                | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-150 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
