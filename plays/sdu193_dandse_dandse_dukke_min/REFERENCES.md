# Dance, Dance, Doll of Mine!: Reference

## Line of Work

Staging Hans Christian Andersen's _Dandse, dandse Dukke min!_ (SDU-193). Den lille Amalie leger og synger med sin dukke, mens Tante prøver at opdrage hende til fornuft.

## Origin

Hans Christian Andersen, _Dandse, dandse Dukke min!_, catalogued as SDU-193 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Dandse, dandse Dukke min!_ (SDU-193)  | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-193 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
