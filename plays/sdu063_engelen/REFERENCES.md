# The Angel: Reference

## Line of Work

Staging Hans Christian Andersen's _Engelen_ (SDU-063). En engel fører et afdødt barn til Himlen, mens de samler minder og en vissen markblomst fra barnets jordeliv.

## Origin

Hans Christian Andersen, _Engelen_, catalogued as SDU-063 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Engelen_ (SDU-063)                    | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-063 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
