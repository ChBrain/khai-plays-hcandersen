# What the Thistle Experienced: Reference

## Line of Work

Staging Hans Christian Andersen's _Hvad Tidselen oplevede_ (SDU-185). En stolt tidsel på marken drømmer om roserne, spises af et æsel og plukkes til sidst som et minde om kærlighed.

## Origin

Hans Christian Andersen, _Hvad Tidselen oplevede_, catalogued as SDU-185 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Hvad Tidselen oplevede_ (SDU-185)     | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-185 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
