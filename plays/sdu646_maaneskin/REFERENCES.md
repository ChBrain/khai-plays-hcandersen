# I Maaneskin: Reference

## Line of Work

Staging Hans Christian Andersen's _Maaneskin_ (SDU-646). Alpescener fra de tyrolske Bjærge om Rosels trofasthed og Mathies' hjemkomst fra krigen under måneskinnet.

## Origin

Hans Christian Andersen, _Maaneskin_, catalogued as SDU-646 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Maaneskin_ (SDU-646)                  | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-646 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
