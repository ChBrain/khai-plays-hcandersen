# It's Quite True!: Reference

## Line of Work

Staging Hans Christian Andersen's _Det er ganske vist!_ (SDU-106). En lille fjer spredes og vokser gennem sladder, indtil den bliver til en historie om fem døde høns.

## Origin

Hans Christian Andersen, _Det er ganske vist!_, catalogued as SDU-106 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Det er ganske vist!_ (SDU-106)        | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-106 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
