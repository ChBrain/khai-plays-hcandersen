# What Old Johanne Told: Reference

## Line of Work

Staging Hans Christian Andersen's _Hvad gamle Johanne fortalte_ (SDU-196). Gamle Johanne beretter om sølle Rasmus, der modløs visner bort, mens trofast omhu og erindring lever videre.

## Origin

Hans Christian Andersen, _Hvad gamle Johanne fortalte_, catalogued as SDU-196 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                        | Scope                                                                 |
| ----------------- | --------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Hvad gamle Johanne fortalte_ (SDU-196) | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk)  | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-196 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
