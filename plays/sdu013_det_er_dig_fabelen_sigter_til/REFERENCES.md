# It is You, the Fable refers to!: Reference

## Line of Work

Staging Hans Christian Andersen's _Det er Dig, Fabelen sigter til!_ (SDU-013). Fabelen søger mennesket for at spejle dets forfængelighed og afsløre den skjulte sandhed.

## Origin

Hans Christian Andersen, _Det er Dig, Fabelen sigter til!_, catalogued as SDU-013 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                            | Scope                                                                 |
| ----------------- | ------------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Det er Dig, Fabelen sigter til!_ (SDU-013) | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk)      | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-013 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
