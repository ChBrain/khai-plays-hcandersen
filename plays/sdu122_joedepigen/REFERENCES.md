# The Jewish Girl: Reference

## Line of Work

Staging Hans Christian Andersen's _Jødepigen_ (SDU-122). Sarah holder sit løfte til sin afdøde moder, men længes i sit indre efter den kristne forløsning.

## Origin

Hans Christian Andersen, _Jødepigen_, catalogued as SDU-122 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Jødepigen_ (SDU-122)                  | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-122 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
