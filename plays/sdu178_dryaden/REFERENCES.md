# The Dryad: Reference

## Line of Work

Staging Hans Christian Andersen's _Dryaden_ (SDU-178). En dryade forlader sit kastanjetræ for at opleve Paris, men hendes frihed koster hende livet.

## Origin

Hans Christian Andersen, _Dryaden_, catalogued as SDU-178 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Dryaden_ (SDU-178)                    | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-178 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
