# The Elves on the Heath: Reference

## Line of Work

Staging Hans Christian Andersen's _Alferne paa Heden_ (SDU-003). Alferne danser på heden under måneskinnet, mens en fattig hyrde iagttager deres mystiske værn om den vilde natur.

## Origin

Hans Christian Andersen, _Alferne paa Heden_, catalogued as SDU-003 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Alferne paa Heden_ (SDU-003)          | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-003 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
