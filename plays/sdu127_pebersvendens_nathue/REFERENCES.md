# The Pepperman's Nightcap: Reference

## Line of Work

Staging Hans Christian Andersen's _Pebersvendens Nathue_ (SDU-127). En ensom pebersvends stille liv og død i København, mens minderne om hans ungdom holdes i live af en gammel nathue.

## Origin

Hans Christian Andersen, _Pebersvendens Nathue_, catalogued as SDU-127 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Pebersvendens Nathue_ (SDU-127)       | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-127 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
