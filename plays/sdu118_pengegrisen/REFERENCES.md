# The Money Pig: Reference

## Line of Work

Staging Hans Christian Andersen's _Pengegrisen_ (SDU-118). En lersparegris fyldt med mønter ser ned på legetøjet, indtil den falder og går i stykker.

## Origin

Hans Christian Andersen, _Pengegrisen_, catalogued as SDU-118 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Pengegrisen_ (SDU-118)                | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-118 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
