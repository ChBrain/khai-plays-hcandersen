# Plan: Leg

## Taxonomy

Drengens plan om at lege på sandstranden.

## Owner

- Owner: [Drengen](persona_drengen.md)

## Direction

At skabe og lege på stranden uden tanke på morgendagen.

## Orders

Hent vand med din spand, byg et sandslot, og se bølgen skylle det bort uden gråd.

## Implementation

Gennemført ved at fylde sin [spand](piece_spand.md) med sand på [stranden](place_stranden.md) og lege ved [havet](place_havet.md).

- [x] Byg dit slot tæt ved [havet](place_havet.md)
- [x] Lad den dybe [forgaengelighed](process_forgaengelighed.md) ske uden gråd
