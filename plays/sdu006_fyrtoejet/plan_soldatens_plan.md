# Plan: Soldatens Plan

## Taxonomy

En **personlig** plan lagt af soldaten for at opnå rigdom, magt og prinsessens hånd.

## Owner

- Owner: [Soldaten](persona_soldaten.md)

## Direction

Soldaten vil bruge fyrtøjets magi til at finansiere sin luksuriøse livsstil, hente prinsessen og beskytte sig selv mod kongens vrede.

## Orders

Soldaten befaler [hundene](persona_hundene.md) at hente penge og bringe ham prinsessen til kroen.

## Implementation

Planen eksekveres i byen og på kroen ved at slå ild på det stjålne fyrtøj.

## Targets

- [x] Dræbe heksen og beholde fyrtøjet som sit eget aktiv
- [x] Bruge hundene til at genvinde sin tabte rigdom på kroen
- [x] Hidkalde prinsessen om natten for at se hendes skønhed
- [x] Tilkalde hundene ved galgen for at kaste dommerne og monarken op i luften
- [x] Overtage tronen og gifte sig med prinsessen
