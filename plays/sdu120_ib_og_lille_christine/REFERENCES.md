# Ib and Little Christine: Reference

## Line of Work

Staging Hans Christian Andersen's _Ib og lille Christine_ (SDU-120). Ib og lille Christine vokser op sammen i skoven, men deres skæbner skilles før de finder forløsning.

## Origin

Hans Christian Andersen, _Ib og lille Christine_, catalogued as SDU-120 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Ib og lille Christine_ (SDU-120)      | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-120 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
