# The Flea and the Professor: Reference

## Line of Work

Staging Hans Christian Andersen's _Loppen og Professoren_ (SDU-200). En luftslipper og hans dresserede loppe fanges af vilde, men undslipper ved list og en stor ballon.

## Origin

Hans Christian Andersen, _Loppen og Professoren_, catalogued as SDU-200 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Loppen og Professoren_ (SDU-200)      | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-200 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
