# Plan: Bedragernes Plan

## Taxonomy

Bedragernes hemmelige plan om at tilegne sig så meget guld og silke som muligt og flygte, før processionen slutter.

## Owner

- Owner: [Bedragerne](persona_bedragerne.md)

## Direction

Bedragerne vil udnytte hoffets og kejserens forfængelighed og frygt for at hente rigdomme ud af skatkammeret.

## Orders

Kræv dyre materialer, foregiv uafbrudt arbejde på de tomme vævestole, og klæd kejseren af før processionen.

## Implementation

Udføres i hemmelighed i [Vævestuen](place_vaevestuen.md) og ved hoffet.

## Targets

- [x] Etablere vævestolene og kræve guld og silke
- [x] Simulere vævning foran de besøgende embedsmænd
- [x] Modtage yderligere midler fra kejseren personligt
- [x] Foregive at sy klæderne og iklæde kejseren dem før processionen
