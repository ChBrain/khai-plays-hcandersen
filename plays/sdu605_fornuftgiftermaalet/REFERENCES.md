# Fornuftgiftermaalet Nr. 2: Reference

## Line of Work

Staging Hans Christian Andersen's _Fornuftgiftermaalet_ (SDU-605). En satirisk, dramatisk drøm på Skagens Rev, hvor personificerede komedier undervises af en streng skolemester.

## Origin

Hans Christian Andersen, _Fornuftgiftermaalet_, catalogued as SDU-605 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Fornuftgiftermaalet_ (SDU-605)        | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-605 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
