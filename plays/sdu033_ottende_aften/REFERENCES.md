# The Eighth Evening: Reference

## Line of Work

Staging Hans Christian Andersen's _Ottende Aften_ (SDU-033). Månen fortæller om en fange i et tårn, der længes efter frihed bag cellens gitter.

## Origin

Hans Christian Andersen, _Ottende Aften_, catalogued as SDU-033 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Ottende Aften_ (SDU-033)              | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-033 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
