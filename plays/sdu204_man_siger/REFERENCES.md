# Folks Say: Reference

## Line of Work

Staging Hans Christian Andersen's _'"Man siger -"!'_ (SDU-204). 'Byens borgere hvisker og spreder rygter om hinanden, baseret på hvad "man siger".'

## Origin

Hans Christian Andersen, _'"Man siger -"!'_, catalogued as SDU-204 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _'"Man siger -"!'_ (SDU-204)           | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-204 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
