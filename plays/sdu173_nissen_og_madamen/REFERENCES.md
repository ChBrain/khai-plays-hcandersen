# The Goblin and the Woman: Reference

## Line of Work

Staging Hans Christian Andersen's _Nissen og Madamen_ (SDU-173). Nissen må vælge mellem madammens grød og gartnerens vers i en strid om poesi og praktisk husholdning.

## Origin

Hans Christian Andersen, _Nissen og Madamen_, catalogued as SDU-173 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Nissen og Madamen_ (SDU-173)          | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-173 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
