# The Pigs: Reference

## Line of Work

Staging Hans Christian Andersen's _Svinene_ (SDU-098). En munter og bidende samtale mellem en ørn på en mødding og en gammel so i stalden om stolthed og rigtigt liv.

## Origin

Hans Christian Andersen, _Svinene_, catalogued as SDU-098 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Svinene_ (SDU-098)                    | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-098 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
