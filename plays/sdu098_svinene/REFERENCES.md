# Svinene: Reference

## Line of Work

Staging Hans Christian Andersen's _Svinene_ (SDU-098). Fortaellerens rejse gennem den svenske skov, de halvvilde svin ved skovvejen og den afsluttende betragtning, indledt af Dickens' anekdote.

## Origin

Hans Christian Andersen, _Svinene_, a chapter of _I Sverrig_ (1851), catalogued as SDU-098 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Svinene_ (SDU-098, from _I Sverrig_)  | The travel sketch the production stages.                              |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged sketch; the full Andersen catalogue and its SDU-098 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
