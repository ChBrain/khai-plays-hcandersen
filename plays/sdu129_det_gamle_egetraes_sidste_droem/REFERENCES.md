# The Old Oak Tree's Last Dream: Reference

## Line of Work

Staging Hans Christian Andersen's _Det gamle Egetræes sidste Drøm_ (SDU-129). Et egetræ drømmer julenat sin skønneste drøm om himlen, mens en vinterstorm knækker dets rødder.

## Origin

Hans Christian Andersen, _Det gamle Egetræes sidste Drøm_, catalogued as SDU-129 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                           | Scope                                                                 |
| ----------------- | ------------------------------------------ | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Det gamle Egetræes sidste Drøm_ (SDU-129) | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk)     | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-129 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
