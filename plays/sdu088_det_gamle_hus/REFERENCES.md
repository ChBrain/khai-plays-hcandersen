# The Old House: Reference

## Line of Work

Staging Hans Christian Andersen's _Det gamle Hus_ (SDU-088). Et forfaldent hus står i kontrast til moderne omgivelser, mens en tinsoldat bringer trøst til dets ensomme beboer.

## Origin

Hans Christian Andersen, _Det gamle Hus_, catalogued as SDU-088 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Det gamle Hus_ (SDU-088)              | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-088 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
