# The Storks: Reference

## Line of Work

Staging Hans Christian Andersen's _Storkene_ (SDU-025). En storkefamilie på et tag plages af drengenes onde sang, men lærer at flyve og tager en retfærdig hævn over dem.

## Origin

Hans Christian Andersen, _Storkene_, catalogued as SDU-025 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Storkene_ (SDU-025)                   | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-025 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
