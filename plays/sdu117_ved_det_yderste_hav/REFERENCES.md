# At the Utmost Sea: Reference

## Line of Work

Staging Hans Christian Andersen's _Ved det yderste Hav_ (SDU-117). En døende sømand under polarnatten finder gennem drømme en forbindelse til sit hjem.

## Origin

Hans Christian Andersen, _Ved det yderste Hav_, catalogued as SDU-117 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Ved det yderste Hav_ (SDU-117)        | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-117 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
