# The Apple: Reference

## Line of Work

Staging Hans Christian Andersen's _Æblet_ (SDU-208). To børn planter en æblekerne i en urtepotte, men tiden bringer glemsel og forgængelighed, mens træet vokser.

## Origin

Hans Christian Andersen, _Æblet_, catalogued as SDU-208 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Æblet_ (SDU-208)                      | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-208 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
