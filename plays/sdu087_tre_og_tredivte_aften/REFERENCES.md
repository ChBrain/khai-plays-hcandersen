# Thirty-Third Evening: Reference

## Line of Work

Staging Hans Christian Andersen's _Tre og tredivte Aften_ (SDU-087). Månen tager afsked med den fattige maler i tagkammeret og lover at huske deres aftener sammen.

## Origin

Hans Christian Andersen, _Tre og tredivte Aften_, catalogued as SDU-087 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Tre og tredivte Aften_ (SDU-087)      | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-087 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
