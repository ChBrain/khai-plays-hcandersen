# Thirty-Second Evening: Reference

## Line of Work

Staging Hans Christian Andersen's _Billedbog uden Billeder_, den to og tredivte aften (Jean Hersholt's numbering: "Thirty-Second Evening"). Gennem stormens flygtige lysglimt ser månen en fange riste et afskedstegn i cellens mur, før vognen fører ham bort.

## Origin

Hans Christian Andersen, _Billedbog uden Billeder_, catalogued as SDU register item 313 at the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history. The staged evening corresponds to Jean Hersholt's English numbering, "Thirty-Second Evening."

| Source            | Key Work / Event                                          | Scope                                                                 |
| ----------------- | --------------------------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Billedbog uden Billeder_, to og tredivte aften (SDU-313) | The tale the production stages.                                       |
| **Jean Hersholt** | English numbering, "Thirty-Second Evening"                | The evening ordinal the production's title anchors to.                |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk)                    | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged evening; the full Billedbog uden Billeder cycle and its SDU-313 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
