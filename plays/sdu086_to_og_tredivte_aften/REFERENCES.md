# Thirty-Second Evening: Reference

## Line of Work

Staging Hans Christian Andersen's _To og tredivte Aften_ (SDU-086). Månen forsøger at kigge frem bag de jagende skyer under en kold og blæsende nat.

## Origin

Hans Christian Andersen, _To og tredivte Aften_, catalogued as SDU-086 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _To og tredivte Aften_ (SDU-086)       | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-086 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
