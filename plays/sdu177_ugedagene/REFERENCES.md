# The Days of the Week: Reference

## Line of Work

Staging Hans Christian Andersen's _Ugedagene_ (SDU-177). Ugedagene mødes til gilde for at finde ud af, hvem af dem der er den vigtigste dag i det evige kredsløb.

## Origin

Hans Christian Andersen, _Ugedagene_, catalogued as SDU-177 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Ugedagene_ (SDU-177)                  | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-177 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
