# Ole Lukoie: Reference

## Line of Work

Staging Hans Christian Andersen's _Ole Lukøje_ (SDU-057). Ole Lukøje besøger Hjalmar hver nat, sprøjter sød mælk i hans øjne og viser drømme, før Døden henter ham.

## Origin

Hans Christian Andersen, _Ole Lukøje_, catalogued as SDU-057 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Ole Lukøje_ (SDU-057)                 | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-057 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
