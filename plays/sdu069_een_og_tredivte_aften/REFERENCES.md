# Thirty-First Evening: Reference

## Line of Work

Staging Hans Christian Andersen's _Een og tredivte Aften_ (SDU-069). En dansebjørn slipper løs fra sin ejer, finder tre børn alene i deres værelse og bliver deres muntre legekammerat med et trægevær, indtil forældrene og bjørnens ejer finder dem.

## Origin

Hans Christian Andersen, _Een og tredivte Aften_, the Thirty-First Evening in Jean Hersholt's numbering of _Billedbog uden Billeder_, catalogued as item 313 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Een og tredivte Aften_ (item 313)     | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue, its item numbering, and Hersholt's evening count live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
