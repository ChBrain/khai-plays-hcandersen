# The Red Shoes: Reference

## Line of Work

Staging Hans Christian Andersen's _De røde Skoe_ (SDU-072). Karen besættes af sine røde sko og tvinges til at danse utrætteligt, indtil hun søger bod og tilgivelse.

## Origin

Hans Christian Andersen, _De røde Skoe_, catalogued as SDU-072 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _De røde Skoe_ (SDU-072)               | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-072 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
