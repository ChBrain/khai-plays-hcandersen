# The Shirt Collar: Reference

## Line of Work

Staging Hans Christian Andersen's _Flipperne_ (SDU-092). En pralende flip frier til alle han møder, ender som klud og genanvendes til papir.

## Origin

Hans Christian Andersen, _Flipperne_, catalogued as SDU-092 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Flipperne_ (SDU-092)                  | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-092 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
