# Ogier the Dane: Reference

## Line of Work

Staging Hans Christian Andersen's _Holger Danske_ (SDU-075). Holger Danske sover under Kronborg og venter på at vågne, mens en billedskærer foreviger ham i en galionsfigur.

## Origin

Hans Christian Andersen, _Holger Danske_, catalogued as SDU-075 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Holger Danske_ (SDU-075)              | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-075 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
