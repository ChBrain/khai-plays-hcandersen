# Twenty-Ninth Evening: Reference

## Line of Work

Staging Hans Christian Andersen's _Ni og tyvende Aften_ (SDU-055). Månen oplyser en rejsende karet foran en kro, hvor kusken venter.

## Origin

Hans Christian Andersen, _Ni og tyvende Aften_, catalogued as SDU-055 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Ni og tyvende Aften_ (SDU-055)        | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-055 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
