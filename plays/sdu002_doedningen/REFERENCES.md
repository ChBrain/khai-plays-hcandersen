# The Dead Man: Reference

## Line of Work

Staging Hans Christian Andersen's _Dødningen_ (SDU-002). Johannes betaler en død mands gæld og får følgeskab af en mystisk hjælper, der beskytter ham på rejsen.

## Origin

Hans Christian Andersen, _Dødningen_, catalogued as SDU-002 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Dødningen_ (SDU-002)                  | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-002 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
