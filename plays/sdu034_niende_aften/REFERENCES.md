# The Ninth Evening: Reference

## Line of Work

Staging Hans Christian Andersen's _Niende Aften_ (SDU-034). Månen ser grønlændernes dans under nordlyset og Den Syge, der vælger havet og isfjeldene som gravsted.

## Origin

Hans Christian Andersen, _Niende Aften_, the Ninth Evening (Hersholt numbering) of _Billedbog uden Billeder_, catalogued as SDU-034 (register vid 34) in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source                   | Key Work / Event                                                          | Scope                                                                    |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-034 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
