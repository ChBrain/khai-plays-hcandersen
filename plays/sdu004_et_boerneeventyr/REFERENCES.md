# A Child's Tale: Reference

## Line of Work

Staging Hans Christian Andersen's _Et Børneeventyr_ (SDU-004). To børn leger i haven og huset, mens kærlighed og tro spirer gennem deres uskyldige leg.

## Origin

Hans Christian Andersen, _Et Børneeventyr_, catalogued as SDU-004 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Et Børneeventyr_ (SDU-004)            | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-004 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
