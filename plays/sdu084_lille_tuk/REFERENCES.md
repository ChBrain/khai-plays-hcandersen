# Little Tuk: Reference

## Line of Work

Staging Hans Christian Andersen's _Lille Tuk_ (SDU-084). En lille dreng passer sin søster frem for lektierne, men i en drøm besøger byerne ham og lærer ham geografien.

## Origin

Hans Christian Andersen, _Lille Tuk_, catalogued as SDU-084 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Lille Tuk_ (SDU-084)                  | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-084 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
