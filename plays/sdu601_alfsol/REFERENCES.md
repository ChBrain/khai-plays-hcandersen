# Alfsol: Reference

## Line of Work

Staging Hans Christian Andersen's _Alfsol_ (SDU-601). Hans Christian Andersens første nordiske tragedie i fem akter om den skønne Alfsol og hendes tragiske skæbne.

## Origin

Hans Christian Andersen, _Alfsol_, catalogued as SDU-601 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Alfsol_ (SDU-601)                     | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-601 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
