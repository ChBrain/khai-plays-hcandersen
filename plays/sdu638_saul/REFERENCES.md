# Kong Saul: Reference

## Line of Work

Staging Hans Christian Andersen's _Kong Saul_ (SDU-638). En operatekst i fem akter om kong Sauls jalousi mod David, et heksebesøg og Samuels åbne profeti.

## Origin

Hans Christian Andersen, _Kong Saul_, catalogued as SDU-638 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Kong Saul_ (SDU-638)                  | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-638 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
