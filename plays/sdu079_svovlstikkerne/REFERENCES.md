# The Little Match Girl: Reference

## Line of Work

Staging Hans Christian Andersen's _Svovlstikkerne_ (SDU-079). En frysende pige tænder svovlstikker nytårsaften, ser visioner og dør i kulden for at forenes med sin bedstemor.

## Origin

Hans Christian Andersen, _Svovlstikkerne_, catalogued as SDU-079 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Svovlstikkerne_ (SDU-079)             | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-079 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
