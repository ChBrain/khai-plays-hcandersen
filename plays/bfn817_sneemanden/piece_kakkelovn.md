# Piece: Kakkelovnen

## Taxonomy

Det glødende varmeapparat af sort støbejern, der står i stuen og modtager brænde.

## Owner

- Project: khai-plays-hcandersen

## Place

[Køkkenet](place_koecken.md) set gennem vinduet fra [gården](place_gaard.md).

## Load Bearing

Den er centrum for sneemandens længsel og repræsenterer den ødelæggende kærlighed.
