# The Metal Pig: Reference

## Line of Work

Staging Hans Christian Andersen's _Metalsvinet_ (SDU-060). En fattig dreng i Firenze oplever en magisk nat på ryggen af metalsvinet, der fører ham ind i kunstens verden.

## Origin

Hans Christian Andersen, _Metalsvinet_, catalogued as SDU-060 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Metalsvinet_ (SDU-060)                | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-060 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
