# The Diving Bell: Reference

## Line of Work

Staging Hans Christian Andersen's _Dykker-Klokken_ (SDU-001). En dykker stiger ned på havbunden i sin klokke og møder en havfrue i det hemmelighedsfulde dyb.

## Origin

Hans Christian Andersen, _Dykker-Klokken_, catalogued as SDU-001 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Dykker-Klokken_ (SDU-001)             | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-001 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
