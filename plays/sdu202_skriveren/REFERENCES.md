# The Penman: Reference

## Line of Work

Staging Hans Christian Andersen's _Skriveren_ (SDU-202). En flittig skriver arbejder med sin fjerpen ved det mørke skrivebord og nedskriver livets begivenheder.

## Origin

Hans Christian Andersen, _Skriveren_, catalogued as SDU-202 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Skriveren_ (SDU-202)                  | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-202 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
