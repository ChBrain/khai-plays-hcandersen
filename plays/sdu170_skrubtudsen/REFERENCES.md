# The Toad: Reference

## Line of Work

Staging Hans Christian Andersen's _Skrubtudsen_ (SDU-170). En lille skrubtudse drømmer om det store lys og begynder en farefuld rejse mod himlen, drevet af sin indre længsel.

## Origin

Hans Christian Andersen, _Skrubtudsen_, catalogued as SDU-170 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Skrubtudsen_ (SDU-170)                | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-170 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
