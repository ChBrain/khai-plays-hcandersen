# A Rose from Homer's Grave: Reference

## Line of Work

Staging Hans Christian Andersen's _En Rose fra Homers Grav_ (SDU-062). En rose ved Homers grav drømmer om udødelighed og flyver ud i verden som erindring i en sangers bog.

## Origin

Hans Christian Andersen, _En Rose fra Homers Grav_, catalogued as SDU-062 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _En Rose fra Homers Grav_ (SDU-062)    | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-062 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
