# The Poor Woman and the Canary: Reference

## Line of Work

Staging Hans Christian Andersen's _Den fattige Kone og den lille Canariefugl_ (SDU-205). En fattig, ensom kone deler sit sidste brød med sin lille kanariefugl, som synger en trøstende sang.

## Origin

Hans Christian Andersen, _Den fattige Kone og den lille Canariefugl_, catalogued as SDU-205 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                                      | Scope                                                                 |
| ----------------- | ----------------------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Den fattige Kone og den lille Canariefugl_ (SDU-205) | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk)                | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-205 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
