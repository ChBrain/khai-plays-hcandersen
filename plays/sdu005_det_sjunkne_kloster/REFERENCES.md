# The Sunken Monastery: Reference

## Line of Work

Staging Hans Christian Andersen's _Det sjunkne Kloster_ (SDU-005). Et kloster synker i søen, mens munken beder og åmanden ser til fra dybet.

## Origin

Hans Christian Andersen, _Det sjunkne Kloster_, catalogued as SDU-005 in the register of the H.C. Andersen Centret at Syddansk Universitet, the authoritative source for the Danish texts, titles, and publication history.

| Source            | Key Work / Event                       | Scope                                                                 |
| ----------------- | -------------------------------------- | --------------------------------------------------------------------- |
| **H.C. Andersen** | _Det sjunkne Kloster_ (SDU-005)        | The tale the production stages.                                       |
| **SDU Centret**   | [hcandersen.dk](https://hcandersen.dk) | The chronological works database the house anchors title and text to. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The register is external**: this warrant answers for one staged tale; the full Andersen catalogue and its SDU-005 numbering live in the SDU register, not here.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
